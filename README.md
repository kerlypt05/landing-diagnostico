# Landing Page — Diagnóstico Situacional Ejecutable (E2E) + Roadmap 90 Días

Landing page estática (HTML5 + Tailwind CSS vía CDN) para la venta del servicio **“Diagnóstico Situacional Ejecutable (E2E) + Roadmap 90 Días”** de Kerly Engels Palacios Tovar.

Incluye:
- Secciones de alto impacto: Hero, Problema/Agitación, Método 4 fases (52 días), Entregables (Bento Grid), Pricing (3 packs), Autoridad, Decisiones requeridas, Mini-FAQ y CTA final.
- CTA de agenda integrado con Google Calendar.
- Botón de descarga de propuesta en PDF (archivo local dentro del repo).
- Diseño responsivo (Mobile First) con estética corporativa (dark + glass) y foco en autoridad B2B.
